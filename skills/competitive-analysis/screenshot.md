# 阶段 2：截图采集

## 强制截图日志

每步必须输出此行。**不输出 = 截图无效。**

```
[N/N] 操作: {动作} | 截图: {文件名} | AI看到: {OCR文字≥3字} | 状态: ✅/❌
```

**AI看到 列禁止为空/无/—/正常。必须写实际识别到的页面文字。**

## 完成标准（4条全过）

```
□ 日志行数 = 预设步骤数  □ 每行 AI看到 ≥3字
□ 无连续2行 AI看到 相同  □ 最后一张是完成页
```

## 异常

- AI看到 锁屏/桌面 → ❌ → "检测到锁屏，请解锁"
- 连续2行 AI看到 相同 → ❌ → "连续截到同一页面，未生效"
- AI看到 为空 → 重截1次 → 仍为空 → "截图不清晰"

## ADB 操作

```
1. adb devices 确认连接（常用: OnePlus 2715c13b, Xiaomi 9a9d526e）
2. uiautomator dump → cat读UI树 → 找bounds → 算中心点 → input tap
3. screencap → pull → AI识别文字 → 输出日志
```
