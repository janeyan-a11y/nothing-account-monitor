# 阶段 3：报告生成

## 总-分结构

**分（逐个竞品）：**
1. 路径还原：`→` 箭头链，每段标截图编号。不确定标`[待确认]`
2. 步骤表：`| 步骤 | 截图编号 | 🔍观察到的元素 | 用户操作 | 阶段 |`
3. 分支场景：触发条件+路径+截图覆盖。未覆盖标`[待确认]`
4. Mermaid流程图：每个节点标截图编号，无截图节点禁止出现
5. 亮点痛点：各≤3条，标注截图来源

**总（跨竞品对比）：**
1. 对比总表：行=维度，列=竞品，每格标来源。无来源填`[待确认]`
2. 分类总结：只总结已确认共性，不推测
3. 设计建议：逐条标对标来源

## 零幻觉原则

1. 每结论有截图依据 2. 不确定标`[待确认]` 3. 区分🔍观察vs💡推测(<10%)
4. 禁止虚构 5. 有疑问就提问 6. 模糊写"不清晰无法确认"
