# 方式2：搜索网上资料

用户选搜索时执行。

## Step 1 — 搜索

```
"{竞品名} {功能名} 操作步骤 截图"
"{竞品名} {功能名} 官方帮助"
"{竞品名} {功能名} 评测"
```

每轮取前5条结果，提取操作路径和截图。

## Step 2 — 截图收集日志

搜索结果中每张相关截图输出一行：

```
[图N] 来源: {URL} | 步骤: {描述} | AI看到: {内容}
```

## Step 3 — 整理为流程序列

```
| 步骤 | 截图 | 来源URL | 🔍内容 |
| 1 | 图1 | https://... | 首页入口 |
```

缺失步骤标注 `[待确认：搜不到该步骤截图]`。

## Step 4 — 标注来源

报告中每张搜索截图标注：`（来源：搜索结果，URL: https://...）`。不得标为"真机实测"。

## Step 5 — 截图覆盖声明

```
| 竞品 | 搜索收集步骤 | 缺失 | 来源 |
本竞品覆盖: N/M — 基于搜索结果
```

→ 完成进入阶段3 `report.md`
